# Qodana Public API
Qodana Cloud and Qodana Self-Hosted provide the Public API that lets you 
create teams and projects using your build pipelines.

## Contents
This repository contains
* `openapi.yaml` - OpenAPI specification of Public API. This file updates everytime when Public API is added or changed.
* `samples/kotlin` - Sample client in Kotlin based on code generation over `openapi.yaml`.

## How to use the API
Steps are described at https://helpserver.labs.jb.gg/help/qodana/qd-11314-public-api/qodana-cloud-release-notes.html#September+2025
